Chipset / Event API

The event API allows programs to access the built-in event system, giving programs access to user input and other useful data!

getEventqueue()

Returns a tuple passing the entire event queue before clearing it

getEventqueue(doClear)

Functions the same as getEventqueue(), with doClear functioning as a toggle for if the event queue should be cleared

queueEvent(eventName, arguments...)

Manually inject an event into the queue next tick

registerEvent(eventName, function)

Before each tick, every instance of the eventName will be passed as a list into the function

registerEvent(function)

Before each tick, every instance of any event will be passed as a list into the function
